A self-hosted home CCTV and gate monitoring system built around a Raspberry Pi, two TAPO C100 cameras, and a TTLock smart gate — all piped through AWS at under $3/month for two cameras.

The Pi streams RTSP through ffmpeg, writing 10-second HLS segments that upload to S3 for live viewing, plus 5-minute MP4 clips retained for 7 days. CloudFront serves the live HLS, clips, and React SPA. Gate events from TTLock's webhook are stored in DynamoDB and trigger SMS via SNS when failed attempts exceed a configurable threshold. The Pi uses an IoT X.509 certificate — no IAM keys on device.
